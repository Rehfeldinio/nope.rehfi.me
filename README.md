# Zeichen-App im Browser

Interaktive Zeichen-App im Browser mit Partikeleffekten. Pinsel, Radierer, Formen, Text, Stempel -- alles dabei. Im Hintergrund laufen Sterne, Schnee, Feuerwerk oder Konfetti.

## Features

- Freihand zeichnen mit einstellbarer Pinselgröße und Deckkraft
- Formen (Linie, Rechteck, Kreis, Dreieck) mit optionaler Füllung
- Text-Tool und Stempel (Emojis)
- Radierer
- Undo / Redo
- Download als PNG, JPG oder SVG
- Partikeleffekte: Sterne, Schnee, Feuerwerk, Konfetti
- Eigene Partikelfarben und Cursor wählbar
- Keyboard Shortcuts (Ctrl+Z, Ctrl+Y, B, E, ...)

## Setup

```bash
npm install
npm run dev
```

Läuft dann auf [localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Canvas API
